# TypeScript - Bootcamp InfraOps

Repositorio con los módulos 1 y 2 del aprendizaje de TypeScript.

## Estructura
typescript/
├── modulo-1/    # Funciones matemáticas con TypeScript básico
└── modulo-2/    # Interfaces, uniones discriminadas y genéricos
## Módulo 1
Funciones de análisis estadístico: media, mediana y filtrado de valores atípicos. Introducción al tipado estricto y manejo de casos límite con `number | null`.

## Módulo 2
Modelado de datos de un sistema universitario con interfaces, unión discriminada `EstadoMatricula` y un servicio genérico `obtenerRecurso<T>` que simula llamadas a una API.

## Cómo ejecutar

```bash
cd modulo-1
npx tsx src/index.ts

cd ../modulo-2
npx tsx src/index.ts
```
